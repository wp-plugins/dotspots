== Description ==

DotSpots enables anyone to update the news and blogs in real-time with dots: mini-blog posts, including videos, pictures, documents and links, that appear in context on all relevant news articles.

The Wordpress plugin for DotSpots enables the DotSpots engine for every page on your site, allowing users to contribute to your site. You can choose to allow DotSpots to add high-quality dots automatically, or to manually approve each dot.

== Installation ==

1. Extract the plugin and copy the `dotspots` folder to your blogs plugin directory, ie: `/wp-content/plugins/`
1. Find the plugins link on your blog
1. Click the "Activate" link to enable it
1. Visit the "DotSpots Configuration" page under Plugins
1. Enter your DotSpots publisher ID
